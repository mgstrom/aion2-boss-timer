# 永恒之塔2 BOSS刷新倒计时

一个专为《永恒之塔2》玩家设计的BOSS刷新倒计时工具，帮助你及时了解BOSS刷新时间和小游戏进场时间。

## 功能特点

### 🎯 核心功能
- **BOSS刷新倒计时**：添加BOSS后自动计算刷新时间，支持多种时间格式识别
- **小游戏提醒**：双数整点进场提醒，提前3分钟和2分钟语音提示
- **智能语音提示**：使用林志玲风格的语音进行提醒
- **置顶功能**：可将窗口保持在最前端，不被其他窗口遮挡
- **剪贴板检测**：复制包含BOSS名称和时间的文本时自动识别并添加
- **BOSS名称识别**：支持多种BOSS名称变体，自动标准化处理

### 🔔 提醒系统
- **BOSS即将刷新**：提前设置的分钟数提醒
- **BOSS已刷新**：BOSS刷新时的提醒
- **小游戏进场**：双数整点前3分钟和2分钟提醒
- **BOSS时间更新**：更新BOSS刷新时间时的提醒

### 📱 界面特点
- **响应式布局**：支持窗口大小调整
- **实时时间显示**：当前时间和下次进场倒计时
- **BOSS列表**：按剩余时间排序，显示详细信息
- **更新提示**：显示BOSS时间的更新时间和方式

## 如何使用

### 快速添加BOSS
1. 在"快速添加BOSS"输入框中输入BOSS名称和剩余时间
2. 例如：`沉默塔爾坦剩餘時間 1 小時 24 分 34 秒`
3. 系统会自动识别并添加到倒计时列表

### 手动添加BOSS
1. 从"选择BOSS"下拉菜单中选择BOSS
2. 点击"添加BOSS"按钮
3. 系统会根据BOSS的固定刷新时间开始倒计时

### 小游戏提醒
- 系统会在双数整点前3分钟和2分钟进行语音提醒
- 例如：2:00进场，1:57和2:01会有提醒

### 置顶功能
- 点击"置顶"按钮可将窗口保持在最前端
- 再次点击可取消置顶

## 技术实现

- **前端框架**：纯HTML5 + CSS3 + JavaScript
- **桌面应用**：Electron.js
- **数据存储**：localStorage
- **语音合成**：Web Speech API
- **打包工具**：electron-builder
